Live Example: https://hono.forgeon.dev/

Prerequisites:

- [Forgeon CLI](https://forgeon.io/docs/cli) installed globally

To develop locally:

```
npm install
forge dev
```

```
open http://localhost:3000
```

To build locally:

```
npm install
forge build
```

To deploy:

```
npm install
forge deploy
```
